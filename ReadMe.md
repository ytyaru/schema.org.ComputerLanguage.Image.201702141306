# このソフトウェアについて

JSON-LDでプログラミング言語をアイコン画像で表示する。

# 開発環境

* Linux Mint 17.3 MATE
* Firefox 42.0
* [構造化データテストツール](https://search.google.com/structured-data/testing-tool?hl=ja)

## 仕様

* [schema.org](https://schema.org)
	* [ComputerLanguage](https://schema.org/ComputerLanguage)
